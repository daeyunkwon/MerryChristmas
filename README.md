# MerryChristmas
> 개발일: 2023-11-27
<br>

## Introduce🌱
- 크리스마스를 기념하는 애니메이션 화면
<br>

## Experiences🌱
- Timer 객체를 이용해 Label에 한글자씩 입력되는 애니메이션 연출해보기
- Label에서 특정 text의 색상 변경해보기
<br>

## Tech Stack🌱
- UIKit
- Storyboard
- SPM
- Lottie
<br>
